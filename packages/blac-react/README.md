# @blac/react

React bindings for the Blac state management library.

This package provides hooks and utilities to integrate Blac seamlessly with your React applications.

## Installation

```bash
npm install @blac/react @blac/core react
# or
yarn add @blac/react @blac/core react
# or
pnpm add @blac/react @blac/core react
```

## Documentation

For detailed usage and API documentation, please refer to the [README in the src directory](./src/README.md) and the main project documentation.

## Features

- `useBloc`: A React hook to subscribe to Blac instances and re-render components on state changes.
- Optimized for performance and selective re-renders.
- Supports dependency tracking.
- Seamless integration with `@blac/core`.
