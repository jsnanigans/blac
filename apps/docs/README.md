# BlaC Documentation

VitePress documentation site for BlaC.

## Structure

```
guide/          Getting started, concepts, introduction
core/           Core library reference (Cubit, registry, watch, plugins, etc.)
react/          React integration (useBloc, tracking, performance)
plugins/        Official plugins (logging, devtools, persistence)
api/            Auto-generated API reference (do not edit manually)
```

The `api/` directory is generated by `pnpm docs:api` using `scripts/generate-api-docs.mjs`. Do not edit those files by hand.

## Development

```bash
pnpm dev
```

## Building

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```

## Regenerating API docs

From the repo root:

```bash
pnpm docs:api
```

This reads `.api.json` files produced by api-extractor and generates markdown in `api/`.
